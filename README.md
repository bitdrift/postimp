# POST IMP
## AI Social Media Manager

- Post Imp is a service that creates and posts to social media accounts on your behalf.
- Customers primarily interface with Post Imp through SMS.
- Creating a post is as simple as texting an image or describing a post to the service.
- Uses the input to compose a draft social media post.
- Draft is shared with the customer for approval before posting.
- Upon customer approval, the post is automatically posted on requested service(s).
- Customers may iterate on a draft requesting revisions via SMS.
- Works only with Instagram currently but more platforms may be added in the future.
- Uses the domain name postimp.com
- To register for the service, a customer can simply text a phone number (tbd) and the service will reply with links that the user can use to complete signup
- Pricing is per post (tbd) but this feature will be added later.
- Service includes a simple website for completing signup and hosting draft content.
- Service uses online AI model for composing draft posts. It analyzes the image and uses profile info to create custom posts.
- Service stores completed posts in a local database to help it track and improve on posts
- Service uses an onboarding process to collect info about the customer that will be used to customize posts.
